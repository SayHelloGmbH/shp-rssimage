Adds the post thumbnail to the RSS feed using the XML tag specified in the [Media RSS Specification](http://www.rssboard.org/media-rss#media-content).

== Installation ==

1. Upload the `shp-rssimage` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
